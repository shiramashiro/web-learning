属性名：formenctype

说明：如果 button 元素的 type 属性值是 submit，formenctype 的属性值表示提交表单到服务器的内容类型。可选值：

  • application/x-www-form-urlencoded: 未指定时的默认值。
  • multipart/form-data: 如果使用type属性的 input 元素设置文件，使用此值。
  • text/plain

button 按钮的 formenctype 属性为 multipart/form-data 时，表示表单中要上传文件。

button 按钮的 formmethod 指定提交表单的 HTTP 请求方式。
