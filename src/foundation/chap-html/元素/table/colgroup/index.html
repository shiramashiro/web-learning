演示 colgroup 效果

说明：colgroup 标签用于对表格中的列进行组合，以便对其进行格式化。通过使用 colgroup 标签，可以向整个列应用样式，而不需要重复为每个单元格或每一行设置样式。

colgroup 的子元素 col 可以指定属性 span，占用多少个列。colgroup 本身也可以使用span，是指运用的 style 样式从第一列占用到第几列。

注意：只能在 table 元素内使用 colgroup 标签。
